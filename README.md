### Prerequisites
- Java 21
- Maven 3.8+

### Running the Application
```bash
# Clone the repository
git clone https://github.com/ajithsolomonsekar/repo-scorer
cd github-repository-scorer

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
```

The application will start on `http://localhost:8080`
